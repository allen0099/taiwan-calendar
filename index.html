🇹🇼 Taiwan Government Holiday Calendar API

自動更新的台灣政府行事曆 API

📋 功能特色

  • 自動從政府資料開放平台抓取所有可用年度的行事曆
  • 智慧過濾：排除 Google 專用版本，優先使用修正版
  • 自動轉換為 JSON 格式，按年月分檔
  • 每月自動更新，確保資料最新
  • 完全開源且免費使用

📖 使用範例

使用 JavaScript Fetch API 取得資料:

// 取得可用的行事曆列表
fetch('Loading.../index.json')
  .then(res => res.json())
  .then(data => console.log(data));

// 取得 2024 年 10 月的行事曆
fetch('Loading.../2024-10.json')
  .then(res => res.json())
  .then(data => console.log(data));
